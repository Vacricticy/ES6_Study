# 介绍
+ js的标准化组织
    - ECMA
+ js的发展(重要版本)
    - ES3.1         2008年
    - ES5           2009年
    - ES6(ES2015)   2015年

+ ES6兼容性
    - IE10+、Chrome、FireFox、移动端、NodeJS

+ ES6新特性
    - 变量
    - 函数
    - 数组
    - 字符串
    - 面向对象
    - promise
    - generator
    - 模块化

+ 编程思想
    - 变量先定义为const，若发现会改变值，才将const改为let
    - 带有on的基本就是回调函数，比如onResloved,onRejected,onclick,onmouseover(on表示当什么什么时候)
